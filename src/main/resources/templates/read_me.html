사용설명서

서버포트 : 8088 (http://localhost:8088/)
Sql 데이터베이스명 : jjs

개인프로젝트 사용설명서
사용된 기술 : 검색창, 스프링 시큐리티, 자바스크립트, 부트스트랩, 자바, 네이버 Api, 카카오 네이버 구글 로그인아웃 api

기능 구현 :
- 로그인 로그아웃을 통한 회원 인증과 회원 판별을 통한 게시글, 댓글 작성수정과 좋아요 기능구현.
- 좋아요 기능 수행 이후 중복 좋아요시 좋아요 취소.
- 기본적인 게시판 기능 및 페이징
- 네이버 책 검색 Api를 활용한 네이버 도서 검색 기능
